# Position

**position的四个值：static，relative，absolute，fixed**

绝对定位：absolute和fixed统称为绝对定位

相对定位：relative

默认值：static



#### static

放到文档流中该出现的位置



#### relative

相对于原来位置移动，元素仍然处于文档流中，不影响其他元素的位置



#### absolute

分类：只设置 **position: absolute** 和 **同时设置了偏移量**

- 只设置 **position: absolute**
  - 影响后面的
- **同时设置了偏移量**
  - 还会影响自己
