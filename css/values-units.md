# Values and Units

CSS values 用 <> 分隔
<color>


### Lengths

- absolute
- relative


Absolute
cm, mm, Q, in pc, pt, pc

relative
em, rem, percentage, vw(viewport width), vh(viewport height)



### Color
1. key word (red, blue, green...)
2. Hexadecimal
    - #02798b
3. rgb function
    - rgb(r, g, b)
4. rgba function
    - rgba(r, g, b, a)
