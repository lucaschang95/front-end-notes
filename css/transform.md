# transform

- 可以多个函数一起用
- 可以使用`transform-origin`这个属性
- 还可以有3D的效果



## 旋转

**单位: deg**

#### `rotate()`

- 在屏幕内转, 正值为顺时针, 负值为逆时针



#### `rotateX()`

- 以横轴(x轴)旋转



#### `rotateY()`

- 以纵轴(x轴)旋转
